# Archiviation (work-in-progress)

Note: This software does not have a stable version yet.

## Usage

```
$ git clone
$ cd archiviation
$ npm install .
$ npm link
$ archiviation init
$ archiviation build
```

## Notice

In order to avoid troubles, do not use any of the following characters in file names:

```

```
